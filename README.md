### 开发环境：
Node.js + MongoDb + Express 4.x

### 如何使用：
* 首先请启动MongoDb服务
* `npm install` 进行依赖模块安装
* 安装完成后，`npm start` 启动应用，浏览器打开 http://localhost:3000 即可使用
